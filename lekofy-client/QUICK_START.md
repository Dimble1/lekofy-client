# Быстрый старт: Система публикации объявлений

## 📋 Что создано

Модуль публикации объявления на React с универсальной формой, поддерживающей разные категории товаров. Система включает:

- ✅ Выбор категории (11 вариантов) с иконками и анимацией
- ✅ Общие поля: название, цена, описание, город, контакт, фото
- ✅ Дополнительные поля, меняющиеся в зависимости от категории (машины, электроника, услуги и др.)
- ✅ Адаптивный и современный интерфейс с Font Awesome иконками
- ✅ Валидацию для всех полей

(Ниже описана детальная реализация на примере категории "Машины", доступна из UI.)

- ✅ **16 полей** для заполнения данных об автомобиле
- ✅ **Font Awesome иконки** для каждого поля
- ✅ **Валидация формы** на клиенте
- ✅ **Загрузка фотографий** с превью (до 10 фото)
- ✅ **Адаптивный дизайн** для мобильных, планшетов, десктопов
- ✅ **Современный интерфейс** с анимациями и цветовой схемой

## 🚀 Быстрый старт (3 шага)

### Шаг 1: Подключите иконки Font Awesome

**Файл**: `lekofy-client/index.html`

Вставьте в `<head>`:
```html
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.0/css/all.min.css" />
```

### Шаг 2: Настройте маршрутизацию

**Файл**: `lekofy-client/src/App.jsx`

```jsx
import PublishAd from './pages/PublishAd';
// ...

function App() {
  return (
    <Routes>
      <Route path="/publish" element={<PublishAd />} />
      {/* остальные маршруты */}
    </Routes>
  );
}
```

### Шаг 3: Запустите приложение

```bash
cd lekofy-client
npm install        # установка зависимостей
npm run dev        # старт dev-сервера
```

Откройте браузер: **http://localhost:5174/** (порт может измениться). Перейдите на `/publish` или нажмите кнопку "Подать объявление" в навигации.

После этого выберите категорию и заполните форму.

## 🔍 Валидация

Форма проверяет:

- **Все поля** - обязательны для заполнения
- **Год** - между 1900 и текущим годом
- **Объем** - положительное число
- **Пробег** - неотрицательное число
- **Цена** - положительное число
- **Описание** - минимум 20 символов
- **Фото** - от 1 до 10 файлов

При ошибке:
- Поле подсвечивается красным
- Появляется сообщение ошибки
- Форма прокручивается к началу

## 🐛 Решение проблем

### Иконки не видны
- Проверьте что Font Awesome подключена в `index.html`
- Очистите кэш браузера (Ctrl+Shift+Delete)

### Стили не применяются
- Проверьте пути к CSS файлам в компоненте
- Убедитесь что файлы находятся в папке `src/styles/`

### Форма не отправляется
- Откройте F12 > Console и посмотрите ошибки
- Проверьте что API endpoint корректный

### Файлы не загружаются
- Максимум 5МБ на файл
- Только JPG и PNG
- Максимум 10 файлов

## 💡 Советы и трюки

### Сохранение черновика
```javascript
// Сохранить в localStorage перед закрытием
window.addEventListener('beforeunload', () => {
  localStorage.setItem('carFormDraft', JSON.stringify(formData))
})

// Загрузить при открытии
useEffect(() => {
  const draft = localStorage.getItem('carFormDraft')
  if (draft) setFormData(JSON.parse(draft))
}, [])
```

### Автозаполнение моделей
```javascript
// Загрузите модели с сервера в зависимости от марки
useEffect(() => {
  if (formData.brand) {
    fetch(`/api/cars/models/${formData.brand}`)
      .then(r => r.json())
      .then(data => setAvailableModels(data))
  }
}, [formData.brand])
```

### Проверка цены
```javascript
// Сравните с средней ценой на рынке
const checkPrice = async (brand, model, year) => {
  const avg = await fetch(`/api/cars/price/${brand}/${model}/${year}`)
  if (avg > formData.price * 1.5) {
    alert('Цена необычно низкая!')
  }
}
```
