# Компонент AddCarForm - Форма добавления объявления об автомобиле

## Описание
Современная, полностью функциональная форма для добавления объявления об автомобиле на сайт объявлений. Компонент включает:

- ✅ Все 16 требуемых полей
- ✅ Иконки Font Awesome для каждого поля
- ✅ Полная валидация формы
- ✅ Загрузка и превью фотографий
- ✅ Адаптивный дизайн (мобильный, планшет, десктоп)
- ✅ Современный и аккуратный дизайн
- ✅ Плавные анимации
- ✅ Обработка ошибок

## Структура полей формы

### 1. Основная информация
- **Марка автомобиля** (brand) - dropdown с 20+ популярными марками
- **Модель** (model) - dropdown (зависит от выбранной марки)
- **Название объявления** (title) - текстовое поле для краткого описания

### 2. Технические характеристики
- **Год выпуска** (year) - численный input (1900 - текущий год)
- **Объем двигателя** (engine_volume) - численный input в литрах
- **Тип двигателя** (fuel_type) - dropdown (Бензин, Дизель, Гибрид, Электро)
- **Коробка передач** (transmission) - dropdown (Механика, Автомат, Вариатор)
- **Привод** (drive_type) - dropdown (Передний, Задний, Полный)
- **Пробег** (mileage) - численный input в км

### 3. Внешний вид и комплектация
- **Цвет** (color) - dropdown с 11 цветами
- **Положение руля** (steering_wheel) - dropdown (Левый, Правый)

### 4. Место учета и условия покупки
- **Страна учета** (country) - dropdown (Кыргызстан, Казахстан, Россия, Армения)
- **Тип расчета** (payment_type) - dropdown (Наличные, Безналичный, Обмен, Рассрочка)

### 5. Цена и описание
- **Цена** (price) - численный input с валютой "сом"
- **Описание** (description) - textarea (минимум 20 символов)

### 6. Загрузка фотографий
- **Изображения** (images) - множественная загрузка (до 10 фото)
- Превью загруженных фотографий с возможностью удаления

## Валидация

Форма проверяет:
- ✅ Все поля обязательны для заполнения
- ✅ Год: между 1900 и текущим годом
- ✅ Объем двигателя: положительное число
- ✅ Пробег: неотрицательное число
- ✅ Цена: положительное число
- ✅ Описание: минимум 20 символов
- ✅ Фото: минимум 1, максимум 10

При ошибке валидации:
- Красное подсвечивание поля с ошибкой
- Сообщение о ошибке под полем
- Автоматическая прокрутка к началу формы

## Функции компонента

### handleInputChange
Обновляет состояние при изменении значения поля и очищает ошибку.

### validateForm
Валидирует все поля формы перед отправкой.

### handleImageUpload
Обрабатывает загрузку изображений, создает превью и проверяет ограничения (максимум 10 фото).

### removeImage
Удаляет загруженное изображение из списка.

### handleSubmit
Обрабатывает отправку формы, проверяет валидацию и готовит данные для отправки на сервер.
